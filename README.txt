Code used in "Characterization of caffeine response regulatory variants in vascular endothelial cells" by Boye et al. (2022). 
The script used for alignment can be found at https://github.com/cakalita/BiT-STARR-seq/blob/main/alignandprep.sh
DESeq_script.R is the script for differential activity analysis
The script to run the initial ASE analysis can be found at https://github.com/cakalita/BiT-STARR-seq/blob/main/bitstarr_base_script.R
Follow-up ASE analysis is done using  manually_calculate_quasar_metaanalysis.R
PWM_Scan.sh contains the script used to scan for motifs
